Detailed Description

Buzzer Driver.

Basic example for using Buzzer driver:

#include "buzzer.h"
static buzzer_t buzzer = BUZZER_INIT_DEFAULT;
int main( void )
{
...
// Initialize Buzzer
buzzer_init(&buzzer);
// Play a melody
buzzer_play_melody(&buzzer_melody);
...
}
